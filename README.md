# Noor – SaaS لإدارة التوصيل (Multi‑tenant)

هذا المستودع يحتوي مشروع **Noor**: باكند NestJS، واجهة Next.js، وتطبيقين أندرويد (مندوب ومجهّز)،
بالإضافة إلى بنية النشر على Ubuntu (Nginx + PM2) وملفات توثيق ودليل التثبيت.

> هذا الإصدار يتضمن هيكلًا جاهزًا للإقلاع (Bootstrapped) مع صفحات عربية (RTL) وموديولات أساسية،
> وقابل للتوسعة لتغطية جميع المواصفات. راجع `docs/INSTALL_GUIDE.md` للتثبيت الفوري.
